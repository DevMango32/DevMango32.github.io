## 언리얼 엔진이란

**언리얼 엔진(Unreal Engine)**은 에픽게임즈(Epic Games)가 개발한 리얼타임 3D 그래픽 제작 툴이다. 게임 개발뿐 아니라 영화 제작, 건축 시각화, 자동차 디자인 등 다양한 분야에서 쓰인다.

## 핵심 용어

| 용어 | 뜻 |
|---|---|
| 리얼타임(Real Time) | 가상 세계가 현실 시간과 동기화되어, 입력이나 환경 변화에 즉각 반응하는 것 |
| 렌더링(Rendering) | 3D 모델을 2D 이미지나 애니메이션으로 변환하는 과정. 조명·질감을 적용해 빛의 반사, 굴절, 그림자를 계산한다 |
| 실시간 렌더링 | 입력·환경 변화에 즉시 반응해 화면을 그려내는 것. 게임 엔진의 핵심 기능 |

## Unreal Engine vs Unity

| | Unity | Unreal Engine |
|---|---|---|
| 언어 | C# | C++ |
| 진입 장벽 | 낮음 — 인터페이스가 쉬워 접근성이 좋다 | 높음 — 고급 기능이 많아 학습량이 필요하다 |
| 강점 | 빠른 개발, 모바일·인디에 강함 | 세부 컨트롤, 현실감 있는 고급 그래픽 |

두 엔진 모두 뚜렷한 장단점이 있으므로, 자신의 기술 수준과 프로젝트 요구사항에 맞춰 선택하면 된다.
