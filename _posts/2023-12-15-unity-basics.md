처음 게임 프로젝트를 하면서 주먹구구식으로 개발하다 보니 정리가 필요하다고 느꼈다. 유니티를 처음 사용할 때 필요한 지식만 아주 간단하게 정리한다.

## 유니티(Unity)

유니티는 유니티 테크놀로지스가 2004년에 개발한 **게임 엔진**이다. 게임 엔진이란 게임 개발에서 자주 쓰는 기능(3D 계산, 그림자, 사운드 등)을 쉽게 사용할 수 있도록 만든 도구다.

## 유니티 에셋(Asset)

유니티는 게임 개발에 필요한 리소스까지 종합적으로 지원한다. 유니티 게임에 사용되는 리소스를 **에셋**이라고 하며, [유니티 에셋 스토어](https://assetstore.unity.com/ko?locale=ko-KR)에서 각종 에셋을 구입해 사용할 수 있다.

## 유니티 에디터 구성

![](/assets/img/unity-basics-1.png)
*▲ 유니티 에디터 화면 구성*

| 화면 | 역할 |
|---|---|
| Scene 뷰 | 게임을 구성하는 메인 화면 |
| Game 뷰 | 게임을 실행했을 때 보이는 화면 확인 |
| Animation 뷰 | 오브젝트의 애니메이션 동작 관리 |
| Hierarchy 창 | Scene 뷰에 배치한 오브젝트 목록 |
| Project 창 | 게임에서 사용하는 리소스 관리 |
| Console 창 | 오류·경고 메시지 출력 |
